## ğŸ”§ Troubleshooting

### If you see "SignaturePad library not loaded":
1. **Refresh the page** (F5 or Ctrl+R)
2. Wait for page to fully load
3. Try signing again

### If signature pad doesn't open:
1. Check browser console for errors (F12)
2. Make sure you're clicking on an existing agreement
3. Refresh the page and try again

1. Wait a moment after clicking "Sign Now"
2. The signature pad takes a moment to initialize
3. Try again after 1 second

### If signature doesn't save:
1. Make sure you actually drew something (pad not empty)
2. Check internet connection
3. Look at the error message for details
4. Try refreshing and signing again
